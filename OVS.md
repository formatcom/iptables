# OVS (Open Virtual Switch)

Es un software de código abierto, diseñado para ser utilizado
como un switch virtual en entornos de servidores virtualizados. 


## Comandos Basicos

### Lista los bridge
~~~
# ovs-vsctl list-br
~~~

### Crear un bridge
~~~
# ovs-vsctl add-br <bridge>
~~~

### Eliminar un bridge
~~~
# ovs-vsctl del-br <bridge>
~~~
